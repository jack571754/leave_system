### check_integration.py
自动化配置检查脚本，检查项包括：
- 依赖包是否安装
- 环境变量是否配置
- Django 配置是否正确
- BPMN 文件是否存在
- 数据库表是否创建
- SpiffWorkflow 客户端是否正常

**使用方法**：
```bash
python docs/check_integration.py
```

## 🚀 快速开始

### 新手入门
1. 阅读 `02-快速参考手册.md` 了解核心概念
2. 运行 `check_integration.py` 检查配置
3. 参考 `01-集成配置指南.md` 进行详细配置

### 问题排查
1. 运行 `check_integration.py` 查看哪些配置有问题
2. 查看 `03-配置总结.md` 中的配置清单
3. 参考 `01-集成配置指南.md` 中的常见问题部分

### 代码参考
直接查看 `02-快速参考手册.md` 中的代码片段
