## 🛠️ 工具使用

### check_integration.py
自动化配置检查脚本，检查以下项目：
1. ✅ 依赖包是否安装
2. ✅ 环境变量是否配置
3. ✅ Django 配置是否正确
4. ✅ BPMN 文件是否存在
5. ✅ 数据库表是否创建
6. ✅ SpiffWorkflow 客户端是否正常

**使用方法：**
```bash
python docs/check_integration.py
```
